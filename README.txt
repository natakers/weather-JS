Виджет погоды
1. Для запуска виджета необходимо запустить файл weather.html  в браузере 
(тестировалось в Google Chrome Версия 77.0.3865.90 и Firefox Quantum 69.0.1 (64-битный)). 
2. Для добавления и удаления городов в нижнем правом углу расположены скрытые кнопки "Add" и "Delete".
3. Города добавляются по названию города.
